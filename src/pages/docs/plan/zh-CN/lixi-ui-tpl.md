# lixi-ui-tpl ui 的界面的模板

## ui 的界面的模板, lixi-ui-tpl 的核心思想， 介绍，规划，初衷。

### 核心思想

1. 更高效，更简单，更规范的开发项目。

2. 每个人都是架构师，不做架构师的前端不是好前端。

3. 为企业或公司提供的一个 ui 框架模板。

### 介绍

1. 是基于 lixi-ui 的一个框架。

2. 拥有者 lixi-ui 的所有功能。项目目录基本一致。

### 规划

1. 将 lixi-ui 中的组件抽离出来

2. 有着详细的文档教程。

3. 同样 还有考虑到类型的判定。

### 初衷

1. 每个人都是架构师，不想做架构师的前端不是好前端。

2. 开发更加的简单化。开发者的能力大幅提升。
